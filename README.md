# clojure_tetris

A basic implementation of Tetris using Clojure. Text based graphics using the clojure-laterna library (http://sjl.bitbucket.org/clojure-lanterna/).

## Installation

Clone the repo:

    git clone https://github.com/apesic/clojure-tetris.git

## Usage

Run using Leiningen:

    lein run
